# Getting Started

## Requirements

- Python 3.11+
- uv 0.11+
- Telegram bot token
- Telegraph access token

## Install

1. `uv venv`
2. `uv sync --frozen --dev`
3. Copy `.env.example` to `.env` and fill in values (PowerShell: `Copy-Item .env.example .env`).

## First run

`uv run python -m app.main`
